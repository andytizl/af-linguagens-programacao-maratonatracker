# Maratona Tracker

Projeto final desenvolvido para a disciplina de Linguagens de Programação (3º Semestre de Análise e Desenvolvimento de Sistemas). 
Consiste em um site completo, moderno, responsivo e funcional focado em filmes.

## Tema
Catálogo e rastreador das melhores e mais recentes filmes. O site atua como um guia para os usuários acompanharem os lançamentos, clássicos e produções em alta no mundo do cinema e do streaming, com um design moderno focado na experiência do usuário.

## Páginas do Projeto
- **Home (`index.html`):** Apresentação do projeto, carrossel de destaques e seção de produções em alta.
- **Galeria (`novidades.html`):** Catálogo de novidades e últimos lançamentos.
- **Contato (`contato.html`):** Área de atendimento ao usuário com formulário de sugestões e mapa de localização da sede.

## Tecnologias Utilizadas
- HTML5 (Semântico)
- CSS3 (Variáveis, Flexbox, Keyframes)
- Bootstrap (Versão 5.3.3)
- JavaScript (Manipulação de DOM)
- GitHub (Versionamento de Código)

## Funcionalidades
- Menu responsivo (Navbar Bootstrap)
- Carrossel Bootstrap com múltiplos destaques
- Galeria de imagens com organização em Grid
- Formulário de contato completo com validações (HTML5 e JS)
- Mapa incorporado via Iframe
- Animações CSS (Hover nos cards e surgimento de seções)
- Funcionalidades JavaScript:
  - Validação e alerta no envio do formulário
  - Efeito visual dinâmico (Hover) nos cards de filmes
  - Botão "Voltar ao Topo" dinâmico ao rolar a página
